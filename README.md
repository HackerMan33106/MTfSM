# MTfSM ‚Äì Scrap Mechanic Mod Translator to 11 Languages
üí° **This tool allows you to translate Scrap Mechanic mods into 11 languages.** Now you can translate mods not only from English to Russian but also into other languages using AI. The program supports most mods, but there may be small inaccuracies since machine translation is used.

üìå **The program supports 11 languages:** Russian, English, Chinese, Spanish, German, French, Portuguese, Italian, Japanese, Korean, and Polish.

‚è≥ **If the program "freezes"**, just wait ‚Äì it might be processing a large mod, which can take some time.

## üîß How the Program Works
When you launch the program (main.exe), it automatically:

1. Finds the Steam installation and the Scrap Mechanic mods folder (steamapps\workshop\content\387990).
2. Checks for the presence of mods ‚Äì if none are found, it shows a message and exits.
3. Creates a Translate mods folder in the same directory where the program is located.
4. Copies the original files from the mod's English folder into a new folder (e.g., Russian or another language, depending on the settings).
5. Translates JSON files (only the title and description fields) from the source language to the selected one.
6. Saves the translated files in the Translate mods folder without changing the original files.

**‚ö†Ô∏è The program processes only JSON files from the Gui/Language/English folder. If the mod doesn't contain such files, it will be skipped.**

## üìñ Instructions for Use
1. Run main.exe ‚Äì the translation process will begin automatically.
2. During the process, the program will ask several questions:
- Choose the translation language (you can select from 11 available languages).
- Whether to translate the description.
- Specify the path to the mods folder if it‚Äôs not automatically found.
3. Wait for the translation to complete ‚Äì once done, the program will ask you to press any key to exit.
4. Open the Translate mods folder ‚Äì translated versions of the mods will be there.
5. Replace the files in the mod folder if necessary ‚Äì by default, the program does not modify the original files.
  
**If the program can't find the mods, make sure there are files in the folder.**

## üîπ Compiling and Modifying the Code
**MTfSM is an open-source project, and you can modify it to suit your needs.**

üíæ To build the .exe from main.py:

1. Open the program folder.
2. Place the modified main.py in the same folder as the compiler.
3. Run the compiler as administrator.
4. After compiling, the .exe file will be moved to the compiler folder, and temporary files will be deleted.
